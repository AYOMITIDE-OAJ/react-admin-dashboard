# React Admin Grid Control 


## Technology used
1. React (Typescript)
2. Material UI 
3. React-Pro-Sidebar
4. React Hooks




### This application consists of 
* Light & Dark Mode
* Contact Information Details
* X-Grid Table
* Filter Menus
* Pagination of Pages
* Sorting Menu
